Вибір раси для ферал друїда:
Туй нема шо думати, чесно. Кіть за оленів Альянс грати хочете - Eльфи темряви.

А кіть за Орду - Табуренчик наш великий, то-ше й для відмідя файна прибавка(5% здоров'я).

Характеристики друида кота:
  • Влучність — якшо не набереш 8% цього показника - рахуй ти лох галімий, бо будеш бити всьо, но не боса.
  • Майстерність — туй всьо просто, маєш 26 штучок? Молодець, а кіть нєт - то один червоний камінь на майстерність в штани, тай такой туди рижий на майстерність + влучнисть(10+10), то в тебе і влучність буде, і 1 штучка майстерності, якраз до капа.
  • Рейтинг пробиття броні — твій кап 100%(1400 штучок), то мош набрати риш як буде в тебе 2 трині(воля та чешка), і остальні речі на рдб(но не кольца!), сотетиками добиваш 100%, айбо диви шо-б ти не переборщив. Пояснюю: Якщо в тебе 1392 рпб - забий хрін діду в сраку, тай гуляй так, а сокет який ти-б витратив на лишні 12 рпб можеш прикрасити камнем на 20 спритності, бде май полєзно.
  • Кріт. удар — туй я єбався задовбувався май много, тобі треба мати 53% без бафів тай форми, то твій кап.
  • Хаста — є ше в нас такий показник, він впливає на шшвидкість атаки, та регенерації енергії. Сокетами не вбиваєм а тільки від шмоту при остальних капах.

Першим ділом збираш влучність, дале кріт, а як з крітом всьо чотко - рпб. Майстерність бде в тя як одінеш т10.3 ноги тай засокетиш їх, айбо то така фігня шо не горит тобі.


Білд ферала:
Не будь утирком, тай прочитай всі-всі кнопочки, які ти тицяєш як прокачуєш білд. Я тобі дам один однісінький скрін з білдом, від якого тобі потрібно відхилятися, але ліпше бде, кіть ти собі сам зробиш білд, так-би мовити під свій стиль гри.

Зачарування для ферал друїда:
  1. Голова - Магічний значок болю, Рицарі Чорного Клинку.
  2. Плечі - Велике накреслення сокири, Сини Ходира.
  3. Плащик - Сувій чар для плаща - спритність V, Накладання чар, або Вишивка у вигляді рукояті меча, якщо є Кравцяна справа.
  4. Нагрудник - Сувій чар для нагрудника - всі характеристики VII, Накладання чар.
  5. Зап'ястя - Сувій чар для наручей - штурм II, Накладання чар.
  6. Кісті рук - Сувій чар для рукавичок - Влучність, Накладання чар, або Гіпершвидкісні прискорювачі, якщо є Інженерне діло.
  7. Пояс - Споконвічна поясна пряжка, Ковацьке діло.
  8. Штани - Накладки для паніжів з крижаної луски, Шкірництво.
  9. Буци - Сувій чар для взуття - льодопрохідник, Накладання чар.
  10. Зброя - Сувій чар для зброї - берсерк, Накладання чар, або для початку Сувій чар для зброї - мангуст, Накладання чар.


Каміньчики для ферал кішки:

Червоні гнізда - до капу рпб Розтрісканий багряний рубін, при капі Вишуканий багряний рубін.


Жовті гнізда - до капу влучності Міцний царський бурштин, при капі Смертоносний аметрин.


Синє гніздо в клятві - Сльоза кошмарів, в остальні гнізда вставляєте Вишуканий багряний рубін.


Особливий сокет - Алмаз жорстокого землеправника.

Професії для ферал друїда:

Туй всьо просто, кращі професії для пве - Кравчатське діло та Інженерне діло, через нашивку на плащ(Вишивка у вигляді рукояті меча), та зачарування кістей рук(Гіпершвидкісні прискорювачі).


Є ще Кувацьке діло з Ювелірним. Перша професія дасть вам 2 гнізда для камінчиків, друга професія - 3 набагато кращі за характеристиками камені, легко добивати кап.



Їжа та зіллячка для ферал друїда:

Для файного кота ходити без риболовчого та приготовницького вміння не файно. Але якщо в вас є багато грошей - їжу можете купувати на аукціоні. На рахунок фласок - тільки аукціон/добрий рл.


"Біс" їжа - Підкопчена дракопіра риба-янгол.
Непогана альтернатива - Рибний бенкет.


"Біс" фласочки нема, я юзаю - Еліксир гуру.
Та є ще така - Настій нескінченної люті.



Адони для ферал друїда:
  1. TellMeWhen - що-б більш легше приглядувати за дотами, бафами, проками.
  2. ComboPointsRedux - чесно говоря я заїбався задовбався піднімати очі до фрейму таргета, то з цим адоном можна винести панельку КП туді, де буде удобно тобі.

Більше нічого критично не потрібно, вообще можна грати без адонів, але це уже для хардкорщиків.
